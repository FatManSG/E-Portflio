High-End PC Build – ML Workstation

Built and tuned a custom ML workstation with Ryzen 9 9950X3D, RTX 5090, and DDR5 RAM. Optimized BIOS, airflow, and validated performance for reliability under heavy loads.

Long Description

I did an end-to-end assembly and tuning of a custom workstation engineered for machine-learning model training and everyday productivity. The system is built around an AMD Ryzen 9 9950X3D CPU paired with an NVIDIA RTX 5090 GPU, 96 GB of DDR5-6400 RAM running at full XMP profiles, on an ASUS ROG Strix X870E-E Gaming WiFi (AM5) motherboard, all powered by a Corsair HXi 1500 W PSU housed in a Lian Li 217 chassis. I optimized cable routing and case airflow for efficient cooling, resolved an initial Windows boot failure via a clean USB reinstall, and enabled maximum memory bandwidth in the BIOS. Finally, I validated rock-solid reliability under sustained full-load conditions using OCCT stress tests. The result is a stable, high-throughput platform that accelerates intensive training jobs and delivers seamless day-to-day performance.

Technical Highlights

  • Ryzen 9 9950X3D + RTX 5090 + 96GB DDR5-6400
  • Custom airflow & cable routing in Lian Li 217
  • XMP profile tuning + BIOS max bandwidth config
  • OCCT stress test passed for rock-solid stability
