LeafSense – Plant Disease Detection

Created a web app using OpenCV and DenseNet to detect plant diseases in seconds. Built to assist farmers with rapid diagnosis and treatment guidance for better crop health.

Long Description

Me and 2 other teammates grouped together to build LeafSense, a responsive web app (HTML/CSS/JavaScript) that empowers farmers to diagnose crop diseases via computer vision. Uploaded leaf images are preprocessed with OpenCV (background masking) before being classified by a DenseNet121 model trained on 20 000+ samples. Results are returned in under 2 s, alongside an educational module detailing each disease and treatment. This solution targets early intervention, helping to mitigate yield losses and bolster food-security efforts in rural communities.

Technical Highlights

  • Frontend: HTML/CSS/JavaScript responsive app
  • Image preprocessing with OpenCV
  • Classification using DenseNet121 (20K+ samples)
  • Sub-2s inference with diagnosis + treatment tips
